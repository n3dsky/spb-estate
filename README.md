# spb-estate
# Анализ рынка недвижимости Санкт-Петербурга

**Определение рыночной стоимости квартир и выявление аномалий**

## Описание проекта

В проекте используется архив объявлений о продаже квартир из сервиса **Яндекс Недвижимость** за несколько лет. Данные охватывают Санкт-Петербург и соседние населённые пункты Ленинградской области.

Цель проекта — научиться определять рыночную стоимость объектов недвижимости. Для этого проведён глубокий исследовательский анализ данных, выявлены ключевые параметры, влияющие на цену, и заложена основа для автоматизированной системы, которая сможет отслеживать аномалии и возможную мошенническую деятельность.

По каждой квартире доступны два типа данных:
- **Заполненные пользователем** (ручные данные)
- **Автоматические** (получены из геосервисов: расстояние до центра, аэропорта, парков, водоёмов и др.)

## Цель проекта

- Провести предобработку и исследовательский анализ данных (EDA)
- Выявить факторы, сильнее всего влияющие на полную стоимость квартиры
- Определить типичные сроки продажи недвижимости
- Рассчитать и ранжировать среднюю цену за квадратный метр по населённым пунктам
- Исследовать зависимость цены от удалённости от центра
- Подготовить данные для будущей системы обнаружения аномалий и мошенничества

## Выполненные работы

### Предобработка данных
- Выделены новые полезные столбцы
- Обработаны и заполнены/удалены пропуски
- Исправлены типы данных
- Удалены аномальные значения

### Исследовательский анализ данных (EDA)
- Выявлены и удалены аномалии в данных
- Определены основные факторы, влияющие на полную стоимость объекта
- Изучено время продажи недвижимости
- Рассчитана и ранжирована средняя цена за квадратный метр в разных городах и населённых пунктах Ленинградской области
- Исследована зависимость цены от удалённости от центра Санкт-Петербурга

## Результаты проекта

В результате выполнения проекта:
- Проведена качественная предобработка данных и созданы новые информативные признаки
- Выявлены ключевые драйверы цены объектов недвижимости
- Проанализировано распределение времени экспозиции объявлений
- Получен рейтинг населённых пунктов по средней цене за м²
- Установлена чёткая зависимость между расстоянием до центра и стоимостью квартиры

Полученные insights позволяют строить автоматизированную систему оценки недвижимости и выявления подозрительных объявлений.
